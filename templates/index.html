About

This tool is meant to be used as a means to condense several long documents into one summary. The generated summary will follow a set format and pull from the original documents based on key words. The produced output document will also contain links or references to the original documents where the information was pulled from.

The purpose of this project is to support community groups in their external reviews. The evaluations from this review typically span three different documents and can be over 200 pages long all together. This tool is meant to be used to cut down on the amount and complexity of reading to make it easier for organizations to interpret and implement the information in order to make themselves better.

How To Use

Select at least one file to upload. File type must be PDF or Microsoft Word. Then, click generate and the application will automatically download your new report summary.
